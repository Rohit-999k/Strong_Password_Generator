#  MKPass – Strong Password Generator

##  Overview

MKPass is a simple Python program that generates a strong and secure password automatically.

The generated password:
- Has a random length between **10–14 characters**
- Contains at least:
  - 1 Uppercase letter
  - 1 Lowercase letter
  - 1 Digit
  - 1–2 Special characters
- Special characters allowed: `! @ # $ % & * _ -`
- Generates a different password every time

  - `random`
